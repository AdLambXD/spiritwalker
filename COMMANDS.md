# 修仙插件命令系统使用指南

## 🎮 玩家命令 (/xiuzhen)

### 基础命令
```
/xiuzhen                    # 显示基本信息和帮助
/xiuzhen stats             # 查看详细修仙状态
/xiuzhen meditate          # 开始/停止打坐修炼
/xiuzhen top [页码]        # 查看修仙排行榜
/xiuzhen info              # 查看境界体系介绍
```

### 交互式功能
- **点击停止打坐**：开始打坐后会出现可点击的按钮
- **快捷菜单**：在状态页面显示交互式操作按钮
- **悬停提示**：鼠标悬停查看详细说明

## 🔧 管理员命令 (/xiuzhenadmin)

### 数据管理
```
/xiuzhenadmin setrealm <玩家> <境界>    # 设置玩家境界
/xiuzhenadmin addexp <玩家> <修为值>    # 添加修为点数
/xiuzhenadmin reset <玩家>              # 重置玩家所有数据
```

### 系统维护
```
/xiuzhenadmin reloadconfig              # 重载配置文件
/xiuzhenadmin backup                   # 创建数据备份
```

## 🎯 命令别名
- `/xz` - 玩家命令简写
- `/xiuxian` - 玩家命令中文别名
- `/xza` - 管理员命令简写
- `/xzadmin` - 管理员命令完整别名

## 🔐 权限节点

### 玩家权限
- `simplexiuzhen.use` - 使用基础修仙命令（默认开启）

### 管理员权限
- `simplexiuzhen.admin` - 使用管理员命令（默认OP拥有）

## 📊 命令详解

### /xiuzhen stats - 详细状态
显示内容：
- 当前境界和段位
- 修为进度条
- 打坐状态
- 属性加成信息
- 交互式操作菜单

### /xiuzhen top - 排行榜
功能特点：
- 分页显示（每页10名）
- 显示境界、修为等关键信息
- 支持指定页码查看

### /xiuzhenadmin setrealm - 境界设置
可用境界：
- `LianQi` - 炼气
- `ZhuJi` - 筑基  
- `JinDan` - 金丹
- `YuanYing` - 元婴

## ⚠️ 注意事项

1. **数据安全**：`/xiuzhenadmin reset` 命令不可逆，请谨慎使用
2. **权限控制**：确保只有可信人员拥有管理员权限
3. **备份建议**：重要操作前建议先执行备份命令
4. **配置热更**：大部分配置支持在线重载，无需重启服务器
