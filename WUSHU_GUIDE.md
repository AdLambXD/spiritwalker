# 武者系统使用指南

## 🎯 系统概述

武者系统是双修炼体系中的重要组成部分，专注于**战斗训练**和**内力修炼**。与修仙系统不同，武者系统通过激烈的战斗来提升实力。

## ⚔️ 核心机制

### 境界体系
武者境界分为四个主要阶段：
- **修炼境界** (Lv.1)：初学 → 入门 → 小成 → 熟练
- **通脉境界** (Lv.2)：初期 → 中期 → 后期 → 圆满
- **筑经境界** (Lv.3)：初期 → 中期 → 后期 → 圆满
- **化元境界** (Lv.4)：初期 → 中期 → 后期 → 圆满

### 内力系统
- **内力上限**：随境界和段位提升
- **恢复机制**：
  - 战斗中：缓慢消耗（每秒约1点）
  - 休息时：较快恢复（每秒约1.5点）
  - 自然状态：缓慢恢复（每秒约0.5点）

## 🎮 基础使用

### 主要命令
```
/wushu                    # 查看武者系统帮助
/wushu train             # 开始/停止战斗训练
/wushu stats             # 查看武者详细状态
/wushu breakthrough      # 尝试境界突破
/wushu info              # 查看武者境界体系
```

### 战斗训练
```
/wushu train             # 开始战斗训练模式
```
- 开始后，参与战斗会获得**武者修为**和**内力**
- 战斗越激烈，获得的经验越多
- 内力会在战斗中缓慢消耗和恢复

### 状态查看
```
/wushu stats
```
显示信息包括：
- 当前武者境界和段位
- 武者修为值
- 内力值和上限
- 训练状态
- 属性加成

## 🥋 武者功法系统

### 功法命令
```
/wushu kungfu list       # 查看可学习的武者功法
/wushu kungfu learn <ID> # 学习指定功法
/wushu kungfu use <ID>   # 使用指定功法
/wushu kungfu info <ID>  # 查看功法详细信息
```

### 功法类型
- **攻击型**：剑心通明、猛虎下山等
- **防御型**：金刚不坏等
- **辅助型**：身法技巧等

### 功法消耗
所有武者功法都需要消耗**内力**：
- 基础功法：10-20点内力
- 中级功法：25-35点内力
- 高级功法：40-50点内力

## 🔺 境界突破

当满足以下条件时可以尝试突破：
1. **修为达标**：达到当前段位要求的修为值
2. **内力充足**：满足突破所需的内力要求
3. **物品准备**：收集突破所需的特殊物品

### 突破命令
```
/wushu breakthrough
```

### 突破物品要求
不同境界需要不同物品：
- **通脉境界**：钻石剑×1 + 铁锭×32
- **筑经境界**：下界合金锭×8 + 末影珍珠×16
- **化元境界**：下界之星×1 + 龙蛋×1

## 📊 成长机制

### 经验获取
- **战斗杀敌**：击败怪物获得武者修为
- **PVP战斗**：与其他玩家战斗获得额外经验
- **训练模式**：开启训练状态时经验加成

### 内力管理
- **战斗消耗**：使用技能时消耗内力
- **自然恢复**：脱离战斗后自动恢复
- **特殊物品**：某些物品可以快速补充内力

## ⚖️ 系统平衡

### 与修仙系统的区别
| 对比项 | 武者系统 | 修仙系统 |
|--------|----------|----------|
| **主要成长方式** | 战斗训练 | 打坐冥想 |
| **资源类型** | 内力 | 灵力 |
| **经验来源** | 战斗杀敌 | 打坐修炼 |
| **突破重点** | 战斗物品 | 稀有材料 |

### 协同效应
- **双修优势**：同时修炼可获得额外属性加成
- **互补发展**：一方疲软时另一方提供支援
- **战术配合**：团队中角色分工明确

## 🏆 进阶玩法

### 专精路线
1. **纯武者路线**：全力发展武者系统
2. **双修平衡**：均衡发展两个系统
3. **主武辅修**：武者为主，修仙为辅

### 战斗策略
- **内力管理**：合理分配内力使用
- **技能连招**：组合不同功法形成连击
- **时机把握**：选择合适的突破时机

### 团队配合
- **坦克角色**：高防御武者承担伤害
- **输出角色**：高攻击武者负责输出
- **辅助角色**：使用辅助功法支援队友

## ⚠️ 注意事项

1. **内力限制**：不要过度消耗内力，注意恢复
2. **突破风险**：突破失败不会损失修为，但需要重新准备物品
3. **平衡发展**：建议与修仙系统协调发展
4. **物品储备**：提前准备突破所需物品
