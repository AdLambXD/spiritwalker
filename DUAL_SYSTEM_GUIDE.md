# 双系统修仙插件 - 内力与灵力系统说明

## 🎯 系统概述

本插件实现了独特的双修炼系统，玩家可以同时修炼**修仙系统**（灵力）和**武者系统**（内力），两大系统相互独立又相辅相成。

## ⚡ 修仙系统（灵力）

### 核心特点
- **境界成长**：炼气 → 筑基 → 金丹 → 元婴 → 化神
- **资源类型**：灵力（Lingli）
- **成长方式**：打坐修炼为主，战斗为辅
- **功法特色**：元素法术、治疗辅助、空间操控

### 灵力机制
- **自然恢复**：打坐时快速恢复，非战斗状态缓慢恢复
- **消耗方式**：施放法术、使用仙术技能
- **上限增长**：随修仙境界和段位提升

### 修仙功法示例
- 🔥 **火球术** - 基础攻击法术，消耗15灵力
- ⚡ **雷击术** - 中级攻击法术，消耗25灵力
- ❤️ **仙灵治愈** - 治疗法术，消耗20灵力

## 💪 武者系统（内力）

### 核心特点
- **境界成长**：修炼 → 通脉 → 筑经 → 化元
- **资源类型**：内力（Neili）
- **成长方式**：战斗训练为主，打坐为辅
- **功法特色**：武术招式、身体强化、武器技巧

### 内力机制
- **自然恢复**：战斗中缓慢消耗，休息时恢复
- **消耗方式**：施展武技、强化身体、使用兵器
- **上限增长**：随武者境界和段位提升

### 武者功法示例
- ⚔️ **剑心通明** - 基础剑法，消耗20内力
- 🛡️ **金刚不坏** - 防御功法，消耗30内力
- 🐅 **猛虎下山** - 终极武技，消耗50内力

## 🔄 双系统互动

### 成长路径差异化
| 对比项 | 修仙系统 | 武者系统 |
|--------|----------|----------|
| **主要成长方式** | 打坐冥想 | 战斗训练 |
| **次要成长方式** | 战斗杀敌 | 静修调息 |
| **资源恢复** | 静止时最快 | 战斗时消耗 |
| **突破要求** | 特殊材料 + 修为 | 稀有物品 + 内力 |

### 协同效应
- **双修优势**：同时修炼两系统可获得额外属性加成
- **资源共享**：部分辅助功法可同时消耗两种资源
- **境界互通**：高境界可解锁跨系统特殊技能

## 🎮 使用指南

### 基础命令
```
/xiuzhen gui              # 打开双系统主界面
/kungfu list             # 查看可学习功法
/kungfu learn <功法ID>   # 学习功法
/kungfu use <功法ID>     # 使用功法
/kungfu info <功法ID>    # 查看功法详情
```

### 修炼方式
**修仙修炼**：
```
/xiuzhen meditate        # 开始打坐（灵力恢复）
```

**武者训练**：
```
/wushu train            # 开始战斗训练（内力提升）
```

### 突破系统
当修为或内力达到要求时：
1. 确保拥有足够的突破物品
2. 使用对应系统突破命令
3. 成功后进入新境界段位

## 📊 资源管理

### 灵力管理
- **最大值**：基础100 + 境界加成 + 段位加成
- **恢复速度**：
  - 打坐：每秒2点
  - 自然：每秒0.5点
  - 战斗：每秒-1点（消耗）

### 内力管理
- **最大值**：基础100 + 境界加成 + 段位加成
- **恢复速度**：
  - 休息：每秒1.5点
  - 自然：每秒0.3点
  - 战斗：视技能而定

## 🔧 配置文件

### martial_realms.yml（武者境界配置）
```yaml
martial_realms:
  XiuLian:                    # 修炼境界
    display_name: "修炼"
    level: 1
    sub_levels:
      - name: "初学"
        required_exp: 0
        required_neili: 0
      - name: "入门"
        required_exp: 100
        required_neili: 50
```

### 功法配置示例
```yaml
fire_ball:
  name: "火球术"
  system_type: "xiuzhen"      # 所属系统
  lingli_cost: 15            # 灵力消耗
  neili_cost: 0              # 内力消耗
  cooldown_ticks: 60         # 冷却时间
```

## 🏆 平衡性设计

### 成长平衡
- **时间投入**：两系统需要不同的专注时间
- **资源稀缺**：突破物品稀有，需要策略获取
- **机会成本**：专精一系vs全面发展各有优势

### PvP平衡
- **修仙优势**：远程法术、控制技能、群体效果
- **武者优势**：近战爆发、生存能力、机动性
- **混合策略**：根据对手选择合适的功法组合

## 🎯 进阶玩法

### 双修路线
1. **均衡发展**：同步提升两系统境界
2. **主副搭配**：主修一系，辅修另一系
3. **专精路线**：全力发展单一系统

### 团队配合
- **治疗组合**：修仙者提供治疗支援
- **坦克输出**：武者承担伤害，修仙者输出
- **控制链**：结合两系统的控制技能
