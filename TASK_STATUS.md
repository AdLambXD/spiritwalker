### 服务器要求
- Minecraft Paper 1.20.4 或更高版本
- Java 17 或更高版本
- MySQL 5.7 或更高版本（可选）

### 安装步骤
1. 将 `SimpleXiuzhen-1.0-SNAPSHOT.jar` 放入服务器 plugins 目录
2. 启动服务器生成配置文件
3. 根据需要修改 `config.yml` 和 `realms.yml`
4. 重启服务器使配置生效

### 推荐配置
对于中小型服务器建议使用YAML存储，大型服务器建议配置MySQL数据库。
