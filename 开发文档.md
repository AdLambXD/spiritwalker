# SimpleXiuzhen 修仙插件 - 开发者指南

## 📋 项目概述

SimpleXiuzhen 是一个功能完整的 Minecraft修仙系统插件，适用于 Paper 1.20.4 服务器。插件实现了双修炼系统（修仙和武者），提供沉浸式的修仙体验。

### 版本信息
- **当前版本**: 1.0-SNAPSHOT
- **支持版本**: Paper 1.20.4+
- **Java 要求**: JDK 17+

**方案一：本地 Maven 安装**
```bash
mvn install:install-file \
  -Dfile=ProtocolLib-5.4.0.jar \
  -DgroupId=com.comphenix.protocol \
  -DartifactId=ProtocolLib \
  -Dversion=5.4.0 \
  -Dpackaging=jar
```

**方案二：运行时检测**

## ⚙️ 核心系统实现

### 1. 双修炼系统

#### 修仙系统（灵力）
- **境界成长**：炼气 → 筑基 → 金丹 → 元婴 → 化神
- **资源类型**：灵力（Lingli）
- **成长方式**：打坐修炼为主，战斗为辅
- **功法特色**：元素法术、治疗辅助、空间操控

#### 武者系统（内力）
- **境界成长**：修炼 → 通脉 → 筑经 → 化元
- **资源类型**：内力（Neili）
- **成长方式**：战斗训练为主，打坐为辅
- **功法特色**：武术招式、身体强化、武器技巧

### 3. 修炼值获取机制

#### 修仙修为获取
```java
// 打坐修炼
if (isMeditating || isRiding) {
    double gain = baseGain * multiplier;  // baseGain = 0.05/sec
    playerData.addExp(gain);
    enhancedPlayerData.addXiuzhenExp(gain);
}
```

#### 武者修为获取
```java
// 战斗训练
if (isInCombatTraining || isInCombat) {
    double gain = baseGain * 0.7 * (isInCombat ? 1.5 : 1.0);
    enhancedPlayerData.addWushuExp(gain);
}
```

#### 怪物击杀奖励
```java
// 基于武者境界的倍数
double realmMultiplier = getWushuRealmMultiplier(currentRealm);
double expReward = baseExp * difficultyMultiplier * realmMultiplier;
enhancedPlayerData.addWushuExp(expReward);
```

### 4. 境界突破系统

#### 修仙境界突破
```java
public boolean breakthrough() {
    if (currentExp >= requiredExp && hasBreakthroughItems()) {
        consumeBreakthroughItems();
        advanceSubLevel();
        applyRealmBonuses();
        return true;
    }
    return false;
}
```

#### 武者境界突破
```java
public boolean wushuBreakthrough() {
    if (currentWushuExp >= requiredWushuExp && 
        currentNeili >= requiredNeili &&
        hasMartialBreakthroughItems()) {
        consumeMartialBreakthroughItems();
        advanceWushuSubLevel();
        applyMartialBonuses();
        return true;
    }
    return false;
}
```

## 🔌 第三方集成

### PlaceholderAPI 集成
```java
public class XiuzhenPlaceholderExpansion extends PlaceholderExpansion {
    @Override
    public String onPlaceholderRequest(Player player, String identifier) {
        if (player == null) return "";
        
        UUID uuid = player.getUniqueId();
        PlayerData data = plugin.getPlayerData(uuid);
        
        switch (identifier.toLowerCase()) {
            case "realm":
                return data.getCurrentRealmKey();
            case "realm_display":
                return data.getCurrentRealmDisplayName();
            case "exp":
                return String.format("%.2f", data.getCurrentExp());
            // ... 更多占位符
        }
        return null;
    }
}
```

```java
// 定期清理离线玩家数据
public void cleanupOfflinePlayers() {
    long now = System.currentTimeMillis();
    Iterator<Map.Entry<UUID, PlayerData>> iterator = 
        playerDataMap.entrySet().iterator();
    
    while (iterator.hasNext()) {
        Map.Entry<UUID, PlayerData> entry = iterator.next();
        if (now - entry.getValue().getLastLoginTime() > 24 * 60 * 60 * 1000) {
            savePlayerData(entry.getKey());
            iterator.remove();
        }
    }
}
```
