# SimpleXiuzhen 管理员命令手册

## 📋 命令概览

所有管理员命令都需要 `simplexiuzhen.admin` 权限节点。

### 命令别名
- `/xiuzhenadmin`
- `/xza`
- `/xzadmin`

---

## 🔧 基础管理指令

### 1. 设置修仙境界
```bash
/xiuzhenadmin setxiuzhenrealm <玩家名> <境界 Key>
```

**功能**：设置玩家的修仙境界

**示例**：
```bash
/xiuzhenadmin setxiuzhenrealm Player1 JinDan
/xiuzhenadmin setxiuzhenrealm Player1 LianQi
```

**可用境界**：
- `LianQi` (炼气)
- `ZhuJi` (筑基)
- `JinDan` (金丹)
- `YuanYing` (元婴)
- `HuaShen` (化神)

**效果**：
- ✅ 自动验证境界是否存在
- ✅ 重置段位到初始等级（0）
- ✅ 重置修为到 0
- ✅ 自动保存数据
- ✅ 管理员和玩家都会收到提示消息


**功能**：查看玩家的完整修仙和武道信息

- 战斗训练状态

## ⚡ 特殊操作指令

### 9. 强制玩家突破
```bash
/xiuzhenadmin forcebreakthrough <玩家名> <类型>
```

**功能**：通过添加大量修为帮助玩家突破境界

**示例**：
```bash
/xiuzhenadmin forcebreakthrough Player1 xiuzhen
/xiuzhenadmin forcebreakthrough Player1 wushu
```

**参数说明**：
- `xiuzhen` - 修仙突破
- `wushu` - 武道突破

**效果**：
- ✅ 添加 999,999 点修为
- ✅ 大概率触发境界突破
- ✅ 自动保存数据
- ✅ 双方收到提示消息

**使用场景**：
- 玩家修为不足但需要快速突破
- 服务器活动奖励
- 管理员特殊奖励

### 11. 重置玩家数据
```bash
/xiuzhenadmin reset <玩家名>
```

**功能**：完全重置玩家的修仙数据

**示例**：
```bash
/xiuzhenadmin reset Player1
```

**效果**：
- ✅ 删除玩家数据文件
- ✅ 从内存中移除数据
- ✅ 重置所有境界和修为
- ✅ 重置所有功法
- ✅ 重置所有属性

**警告**：
- ⚠️ **此操作不可逆！**
- ⚠️ 玩家会收到通知消息
- ⚠️ 仅在线玩家有效

**使用场景**：
- 玩家主动申请重玩
- 数据异常需要重置
- 违规行为处罚

## ⚠️ 注意事项

### 权限控制
- 所有管理员命令都需要 `simplexiuzhen.admin` 权限
- 建议仅授予信任的管理员
- 不要给予普通玩家

### 数据备份
- 执行重大操作前务必备份数据
- 定期执行备份操作
- 保留多个历史备份版本

### 玩家沟通
- 执行操作前通知玩家
- 解释操作原因和影响
- 记录操作日志以备查证

### 谨慎使用
- `reset` 和 `clearkungfu` 会删除玩家数据
- 使用前务必确认玩家和操作的准确性
- 建议先在测试环境验证

## 📝 TAB 补全说明

所有命令都支持 TAB 智能补全：

1. **命令补全**：输入 `/xza` 后按 TAB 显示所有命令
2. **玩家补全**：输入玩家名时按 TAB 自动补全在线玩家
3. **参数补全**：
   - 境界名称：TAB 补全可用境界
   - 数值选项：TAB 显示常用数值（100, 500, 1000, 5000, 10000）
   - 突破类型：TAB 显示 `xiuzhen` 或 `wushu`
