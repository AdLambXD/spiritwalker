# 修仙插件GUI系统使用指南

## 🎮 新增GUI功能

本次更新为SimpleXiuzhen修仙插件添加了完整的图形用户界面系统，让玩家能够通过直观的界面来管理和体验修仙系统。

## 📋 可用命令

### 主要命令
```
/xiuzhen gui        - 打开图形界面（推荐新手使用）
/xiuzhen            - 查看基本信息和帮助
/xiuzhen stats      - 查看详细修仙状态
/xiuzhen meditate   - 开始/停止打坐修炼
/xiuzhen top        - 查看服务器排行榜
/xiuzhen info       - 查看境界体系介绍
```

## 🖥️ GUI界面功能

### 主修仙界面
使用 `/xiuzhen gui` 命令打开主界面，包含以下功能区域：

**중앙 상태面板**
- 显示当前境界和修为
- 实时显示修炼状态
- 点击可查看详情

**左侧功能区**
- 🟢 **开始打坐** - 点击开始修炼获得修为
- 📘 **详细状态** - 查看完整的修仙信息
- 🏆 **修仙排行** - 查看服务器排名

**右侧功能区**
- 📚 **功法系统** - 学习和使用各种功法
- ⭐ **境界介绍** - 了解各个境界特点
- 🟥 **关闭界面** - 退出GUI

### 功法系统界面
从主界面点击"功法系统"进入：

**功能特点**
- 显示所有可学习的功法
- 按类型分类展示（攻击/防御/辅助/治疗）
- 显示学习条件和要求
- 彩色标识不同类型功法

**功法类型**
- 🔴 **攻击型** - 火球术、雷击术等
- 🔵 **防御型** - 能量护盾等
- 🟢 **辅助型** - 疾风步等
- 🟣 **治疗型** - 治愈脉冲等

### 境界信息界面
从主界面点击"境界介绍"进入：

**显示内容**
- 炼气境界信息
- 筑基境界信息
- 金丹境界信息
- 元婴境界信息

## 🎯 使用建议

### 新手玩家
1. 使用 `/xiuzhen gui` 打开图形界面
2. 点击"开始打坐"开始修炼
3. 通过界面查看自己的进度
4. 解锁功法后在界面中学习使用

### 进阶玩家
1. 直接使用命令快速操作
2. 通过GUI界面管理功法
3. 查看排行榜激励自己进步

## 🐛 常见问题

**Q: GUI界面打不开怎么办？**
A: 确保你是在线玩家，检查是否有相应权限

**Q: 点击按钮没有反应？**
A: 检查是否满足使用条件（如境界、修为要求）

**Q: 如何关闭GUI界面？**
A: 点击红色"关闭界面"按钮或按ESC键

**Q: GUI会影响游戏性能吗？**
A: 不会，GUI系统经过优化，对游戏性能影响极小
