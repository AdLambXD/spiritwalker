# SimpleXiuzhen 修仙插件 - 使用指南

## 📖 欢迎进入修仙世界

SimpleXiuzhen 是一个功能完整的 Minecraft修仙系统插件，为您提供沉浸式的修仙体验。本指南将帮助您快速上手并掌握修仙的奥秘。

## 🚀 快速开始

### 1. 安装插件

**自动安装（推荐）**：
- 从服务器插件管理器直接安装
- 或联系服务器管理员安装

**手动安装**：
1. 下载最新版本的 `SimpleXiuzhen-*.jar`
2. 将文件放入服务器 `plugins` 目录
3. 重启服务器

### 2. 基础命令

```bash
# 修仙系统
/xiuzhen          # 查看个人信息和帮助
/xiuzhen gui      # 打开图形界面（推荐新手）
/xiuzhen meditate # 开始/停止打坐修炼
/xiuzhen stats    # 查看详细修仙状态
/xiuzhen top      # 查看排行榜
/xiuzhen info     # 了解境界体系

# 武者系统
/wushu            # 查看武者系统帮助
/wushu train      # 开始/停止战斗训练
/wushu stats      # 查看详细武者状态
/wushu breakthrough # 尝试武者境界突破

# 功法系统
/kungfu list      # 查看可用功法
/kungfu learn <ID> # 学习功法
/kungfu use <ID>   # 使用功法
/kungfu info <ID>  # 查看功法详情
```

## 🧘‍♂️ 修仙系统

### 境界体系

修仙之路分为五个大境界，每个境界包含多个小段位：

| 境界 | 段位示例 | 特殊能力 |
|------|----------|----------|
| **炼气期** | 一层 → 九层 | 基础修炼 |
| **筑基期** | 初期 → 后期 | 属性提升 |
| **金丹期** | 一转 → 九转 | 战斗加成 |
| **元婴期** | 初婴 → 大婴 | 高级技能 |
| **化神期** | 入神 → 大成 | 神通广大 |

### 修炼方式

#### 1. 传统打坐 🧘
```bash
/xiuzhen meditate  # 开始/停止打坐
```
- 保持静止获得基础修为
- 移动距离过大将中断修炼
- 适合喜欢安静修炼的玩家

#### 2. 盔甲架冥想 ⚔️
```bash
# 生成冥想盔甲架
/summon armor_stand ~ ~ ~ {Invisible:1b,NoBasePlate:1b}
# 骑乘上去自动进入冥想状态
```
- 骑乘盔甲架自动进入冥想状态
- 享受**2 倍修为增长**加成
- 更加舒适自然的修炼体验

#### 3. 战斗修行 ⚔️
- 击杀怪物获得修为奖励
- 不同境界享受不同倍数加成
- 实战中提升修为两不误

### 灵力管理

**灵力恢复速度**：
- 打坐时：每秒 2 点
- 自然状态：每秒 0.5 点
- 战斗时：每秒 -1 点（消耗）

**灵力用途**：
- 施放法术
- 使用仙术技能
- 施展特殊能力

## 💪 武者系统

### 境界体系

武者之道分为四个大境界：

| 境界 | 段位示例 | 特点 |
|------|----------|------|
| **修炼境界** | 初学 → 熟练 | 基础武术 |
| **通脉境界** | 初期 → 圆满 | 经脉通畅 |
| **筑经境界** | 初期 → 圆满 | 经络坚固 |
| **化元境界** | 初期 → 圆满 | 元气化形 |

### 修炼方式

#### 1. 战斗训练 ⚔️
```bash
/wushu train  # 开始战斗训练
```
- 参与战斗获得武者修为
- 内力在战斗中缓慢消耗和恢复
- 越激烈的战斗获得的经验越多

#### 2. 怪物击杀 🎯
- 击杀怪物获得武者修为
- 武者境界越高，获得的奖励越多
- 适合喜欢冒险的玩家

### 内力管理

**内力恢复速度**：
- 休息时：每秒 1.5 点
- 自然状态：每秒 0.3 点
- 战斗时：视技能而定（消耗）

**内力用途**：
- 施展武技
- 强化身体
- 使用兵器技巧

| 类型 | 图标 | 示例 | 特点 |

| **攻击型** | 🔴 | 火球术、雷击术 | 造成伤害 |


**常见功法**：
- 🔥 **火球术** - 基础攻击法术，消耗 15 灵力
- ⚡ **雷击术** - 中级攻击法术，消耗 25 灵力
- ❤️ **仙灵治愈** - 治疗法术，消耗 20 灵力

### 武者功法示例

**常见功法**：
- ⚔️ **剑心通明** - 基础剑法，消耗 20 内力
- 🛡️ **金刚不坏** - 防御功法，消耗 30 内力
- 🐅 **猛虎下山** - 终极武技，消耗 50 内力

### 功法要求

学习和使用功法需要满足：
- **境界要求**：必须达到指定境界

## 🔺 境界突破

### 修仙突破

当修为达到要求时：
1. 确保拥有足够的突破物品
2. 使用 `/xiuzhen breakthrough`（如有此命令）
3. 成功后进入新境界段位

**突破物品示例**：
- 炼气→筑基：下界合金锭×4 + 末影珍珠×8
- 筑基→金丹：下界之星×1 + 龙蛋×1

### 武者突破

当武者修为和内力达到要求时：
1. 收集突破所需的特殊物品
2. 使用 `/wushu breakthrough`
3. 成功后进入新境界段位

**突破物品示例**：
- 修炼→通脉：钻石剑×1 + 铁锭×32
- 通脉→筑经：下界合金锭×8 + 末影珍珠×16
- 筑经→化元：下界之星×1 + 龙蛋×1

## ⚙️ 配置文件

### 修改配置

编辑 `plugins/SimpleXiuzhen/config.yml`：

```yaml
settings:
  cultivation:
    base_gain_per_second: 0.05      # 基础修为增长速度
    move_distance_threshold: 0.5    # 打坐移动阈值
    
    armor_stand_meditation:
      enabled: true                 # 是否启用盔甲架冥想
      multiplier: 2.0               # 冥想增益倍数
  
  mob_kill:
    enabled: true                   # 是否启用怪物击杀奖励
    difficulty_multiplier: 1.0      # 难度倍数
    show_exp_message: true          # 是否显示提示消息
  
  data:
    storage_type: "YAML"            # 存储类型：YAML 或 MYSQL
```

### 重载配置
```bash
/xiuzhenadmin reloadconfig  # 热重载配置（无需重启）
```

## 🌍 PlaceholderAPI 支持

如果服务器安装了 PlaceholderAPI，可以使用以下占位符：

### 基础信息
- `%simplexiuzhen_realm%` - 当前境界 Key（如 `LianQi`）
- `%simplexiuzhen_realm_display%` - 境界显示名称（如 `炼气`）
- `%simplexiuzhen_sublevel_name%` - 当前段位名称（如 `三层`）

### 修为相关
- `%simplexiuzhen_exp%` - 当前修为值（如 `1234.56`）
- `%simplexiuzhen_formatted_exp%` - 格式化修为值（如 `1.23K`）
- `%simplexiuzhen_next_level_exp%` - 下级所需修为

### 状态信息
- `%simplexiuzhen_is_meditating%` - 是否正在打坐
- `%simplexiuzhen_meditation_status%` - 打坐状态文字
- `%simplexiuzhen_status_icon%` - 状态图标

### 使用示例

**在聊天中显示**：
```

```

**在记分板中使用**：

```yaml
    - "&a境界：%simplexiuzhen_formatted_realm%"
```

## 🐛 常见问题

### Q: GUI 界面打不开怎么办？
A: 确保你是在线玩家，检查是否有相应权限。如果问题持续，尝试使用命令代替 GUI。

### Q: 点击按钮没有反应？
A: 检查是否满足使用条件（如境界、修为要求）。某些功能需要达到特定境界才能解锁。

### Q: 如何关闭 GUI 界面？
A: 点击红色"关闭界面"按钮或按 ESC 键。

### Q: 修炼状态异常怎么办？
A: 检查配置中的检测参数是否合理，或联系服务器管理员调整。

### Q: 数据保存失败？
A: 确认配置文件格式正确，检查存储路径权限。如果使用MySQL，检查数据库连接是否正常。

### Q: 命令无响应？
A: 验证玩家权限节点 `simplexiuzhen.use` 是否已分配。

### Q: 插件无法加载？
A: 检查 Java 版本（需 17+）和 Paper 版本兼容性。查看服务器日志获取详细错误信息。

## ⚠️ 注意事项

1. **内力限制**：不要过度消耗内力，注意恢复
2. **突破风险**：突破失败不会损失修为，但需要重新准备物品
3. **平衡发展**：建议与修仙系统协调发展
4. **物品储备**：提前准备突破所需物品
